# PyTeach

**PyTeach** is a self-contained desktop learning platform for Python test automation. It guides students through 70 structured lessons across 8 modules — from Python fundamentals to Playwright, API testing, mobile testing, and professional tooling — entirely offline, with no cloud dependencies.
